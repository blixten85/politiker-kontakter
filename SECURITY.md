# Säkerhetspolicy

## Omfattning

Det här projektet är en skrapare som hämtar publikt publicerade
e-postadresser till förtroendevalda från kommuners och regioners egna
webbplatser. Projektet hanterar inga inloggningsuppgifter, personuppgifter
utöver redan offentligt publicerad kontaktinformation, eller känslig data.

## Rapportera en säkerhetsbrist

Om du upptäcker en säkerhetsbrist (t.ex. en sårbarhet i hur skraparen
hanterar indata, beroenden, eller Docker-konfigurationen), rapportera den
privat via [GitHub Security Advisories](../../security/advisories/new) för
detta repo, istället för att öppna ett publikt issue.

Vi återkommer så snart vi kan med en bekräftelse och, om relevant, en plan
för åtgärd.

## Beroenden

Säkerhetsuppdateringar av tredjepartsbibliotek (Playwright, pypdf, m.fl.)
hanteras via Dependabot. Öppna gärna ett issue om du ser en känd CVE som
inte redan flaggats.
